Modular Layout System

This page demonstrates the new modular layout system with dynamically loaded navigation and footer components.

How It Works

Add Container Divs

Simply add <div id="navbar"></div> and <div id="footer"></div> to your HTML.

Include Layout Script

Add <script src="assets/js/layout.js"></script> before closing body tag.

Automatic Loading

The script automatically loads and configures the navigation and footer components with proper active states.

Features

  • Automatic component loading
  • Smart active navigation detection
  • Subdirectory path handling
  • Mobile menu functionality
  • Error handling and fallbacks
  • No external dependencies
